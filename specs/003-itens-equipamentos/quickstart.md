# Quickstart: Itens e Equipamentos

Este guia descreve rapidamente como os dados de itens e equipamentos foram modelados.

1. Navegue até a pasta `docs/itens/` (que será criada na fase de implementação).
2. Lá você encontrará 4 tabelas base em 4 arquivos distintos:
   - `armas.md`: Estatísticas de dano e preços para o combate ofensivo.
   - `armaduras.md`: Regras de CA e restrições de furtividade/Força para defesa.
   - `utilizaveis.md`: Itens comuns de equipamento sem uso consumível.
   - `consumiveis.md`: Itens como ração, bálsamo, água que se esgotam com o uso.

O sistema baseia-se na **Ausência de Magia Arcana e Itens Mágicos**, utilizando o paradigma de Shekels de Prata (SP) para trocas.
