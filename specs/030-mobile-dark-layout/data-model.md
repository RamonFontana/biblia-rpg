# Data Model: Mobile & Dark Mode Layout

Esta feature não altera modelos de banco de dados ou tipos principais. As alterações são restritas à camada de visualização (View Layer) e aos tokens de design do CSS.

## Tokens de Design & Tema
O sistema usará as classes do Tailwind para consolidar a interface:
- **Base (Background):** Cores extremamente escuras como `bg-slate-900` a `bg-slate-950` para o fundo da página, assim como já é feito na Criação de Personagem.
- **Superfícies (Cards, Modais):** `bg-slate-800`.
- **Textos Principais:** `text-gray-100` ou `text-slate-200`.
- **Bordas e Divisórias:** `border-slate-700`.

Nenhum dado relacional foi modificado.
