# Quickstart: NPC Session Management

To test this feature locally:
1. Ensure the database migration has run to add `is_npc` to `characters`.
2. Login as a Game Master and open a session dashboard.
3. Use the "Add NPC" button to insert a character marked as `is_npc = true`.
4. Click on the NPC in the roster to open their character sheet and interact with their inventory.
