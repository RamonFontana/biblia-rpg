# Quickstart: Mostrar Dano de Armas no Inventário

Não há novas dependências, comandos de inicialização ou configurações de ambiente para esta feature.

O desenvolvimento ocorrerá unicamente em `src/components/inventory/InventoryList.tsx`.

Para testar localmente:
1. Suba o servidor com `npm run dev`.
2. Abra a interface de uma sessão ativa ou inventário de personagem.
3. Observe se as armas possuem agora a indicação de dano (ex: "Dano: 1d6 cortante") e se armaduras não exibem falsamente propriedades de dano.
